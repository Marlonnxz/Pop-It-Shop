¿Qué es Pop It?

Pop It se traduce literalmente como "hazlo estallar". Es un juego infantil, cuyo proceso se puede comparar con explotar plástico de burbujas de embalaje. Pero a diferencia del plástico, los Pop It se pueden presionar con sensaciones y sonidos característicos "infinitamente", ya que decenas de burbujas de silicona simplemente se empujan hacia el otro lado.

Reglas del juego

Los jugadores se turnan y presionan las burbujas (una o varias) en una fila. Se pueden presionar como máximo 3 burbujas a la vez, y deben ser adyacentes. No se pueden presionar varias burbujas si no están juntas. ¡El objetivo es hacer que el oponente realice el último movimiento. Quien hace el último movimiento pierde!
